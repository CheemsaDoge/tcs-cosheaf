# Operator Skill Package

This document points to the portable Cosheaf operator instruction bundle:

```text
docs/skills/cosheaf-operator/SKILL.md
```

The package is documentation only. It is not runtime authority, not an MCP
transport, not a hosted-provider integration, and not a replacement for the
repository rules in `AGENTS.md`, validation, gates, human review, verifier
evidence, or accepted promotion.

Use it when a ChatGPT/Skill-like environment needs concise operator
instructions for TCS-Cosheaf work. The skill keeps CLI as the preferred path,
MCP as an optional adapter, and all write surfaces limited to existing
draft/proposal/review-context/runtime records.

The authoritative workflow details remain:

- `docs/CODEX_WORKFLOW.md`
- `docs/EXTERNAL_OPERATOR_RUN_LOOP.md`
- `docs/OPERATOR_WORKSPACE_DEMO.md`
