# Reviewable Research Workflows

This file is a stable entry point for the V14 reviewable-workflow line.
Operational details live in:

- [Reviewable workflows](WORKFLOWS.md)
- [Draft proposals](DRAFT_PROPOSALS.md)

The current workflow path is:

```text
issue -> workflow runtime -> readiness -> draft proposal -> future review handoff
```

Workflow runtime records and draft proposals are review context only. They do
not create proof, source metadata, human review, verifier pass, gate pass,
accepted status, accepted refutation, or promotion authority.
