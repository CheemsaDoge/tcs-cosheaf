# v0.8.0 - Deterministic Execution Kernel + Librarian + FSM

Status: published. Package metadata records 0.8.0.

Delivers the full deterministic execution kernel:
- **Librarian**: policy-scored deterministic retrieval that ranks artifacts by
  text relevance, quality prior, freshness, and status-aware memory partitioning
  (hot/warm/cold). Never creates claims or accepted knowledge.
- **Orchestrator FSM**: durable state machine (created -> planned -> ready ->
  running -> finalized/blocked/failed/abandoned) with validated transitions,
  replay log, and next-action recommendations.
- **Local action registry**: 14 whitelisted safe Cosheaf actions, deterministic
  worker profiles (librarian/planner/checker/handoff).
- **Non-dry-run loop execution**: bounded research-loop can execute whitelisted
  local actions through --execute-local-actions.
- **Memory feedback**: deterministic weight updates from execution outcomes.

No arbitrary shell, network, hosted provider defaults, accepted writes, human
review, auto theorem proving, or promotion authority.
