# v0.8.0 - Deterministic Worker Loop + Local Action Registry

Status: release candidate. Package metadata records 0.8.0.

This release adds a deterministic local action registry, bounded non-dry-run
research-loop execution for whitelisted local actions, and deterministic worker
profiles. 14 safe local actions are callable through cosheaf action CLI.
Non-dry-run loop execution is now possible through --execute-local-actions.

No arbitrary shell, network, hosted provider, accepted writes, human review, or
promotion authority is added. Loop success never means accepted status.
