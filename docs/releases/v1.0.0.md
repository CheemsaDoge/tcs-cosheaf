# v1.0.0 - AI Math Collaborator MVP

Status: release candidate. Package metadata and `cosheaf.__version__` record
`1.0.0`. The public tag, GitHub release, post-tag release smoke, and
downstream workspace-template/public-KB pin updates are not complete until the
publication closeout task lands.

This release packages the V18 AI Math Collaborator MVP line on top of the
published `v0.12.0` Research Memory Learning + Benchmark Suite v1 baseline.

## Included Surfaces

- CLI-first workspace validation, gates, indexing, context packs, lifecycle
  governance, and promotion readiness.
- Reviewable workflow, checker, gap, research-loop, research-run, campaign,
  memory, benchmark, comparison, and static-report surfaces.
- Operator session and review handoff surfaces for external human or AI
  operators.
- Fake/default-off provider paths and optional verifier/tool adapters under
  their existing explicit policies.
- Workspace-template canonical `make ai-math-collaborator-demo` path.
- V18 authority-boundary, public/private policy, operator, benchmark, and
  release-audit documentation.

## Authority Boundary

Cosheaf is a safe research harness with reviewable outputs, not an autonomous
source of accepted truth. Workflow, campaign, operator, checker, benchmark,
comparison, report, memory, and handoff outputs are review context or sidecar
guidance only.

They are not proof, source metadata, human review, verifier pass, gate pass,
accepted status, accepted theorem/refutation status, or promotion authority.
Skipped, unsupported, unavailable, and inconclusive rows are not passes.

This release candidate does not add automatic theorem proving, Lean/mathlib/
CSLib semantic alignment, AI-as-human-review, automatic accepted promotion,
default hosted provider calls, model training, web UI, production SaaS, or
multi-user permissions.
