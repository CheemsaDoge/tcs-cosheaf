# v0.9.0 - Reviewable Research Workflow MVP + Benchmark Harness

Status: release candidate. Package metadata records 0.9.0.

Adds the cosheaf workflow CLI that wires librarian, orchestrator FSM, action
registry, and research-loop primitives into a single deterministic reviewable
research workflow. Every step records inputs, outputs, status, and authority
notice. Workflow records, draft proposals, and review handoffs remain review
context only. No accepted writes, human review creation, or promotion authority.
